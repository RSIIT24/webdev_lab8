About This Project

This page describes the purpose of the navigation project. The goal is to demonstrate how CSS can transform a simple list of links into structured navigation menus.

The top navigation bar is centered across the page and spans the entire width of the browser window. The sidebar navigation remains fixed on the right side of the screen so it remains visible as users scroll.
